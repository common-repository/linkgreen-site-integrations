The official LinkGreen plugin. Allows sellers on the LinkGreen platform to display items in a carousel using shortcodes.

== Description ==

Create a powerful user experience for your customers, while using your website with the LinkGreen WordPress plugin. Easy to set up using shortcodes, to instantly increase exposure for your product catalogs and have real-time updates sync to your website.

== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin
2. Upload the 'linkgreen-sites' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and activate the plugin

= Install Via Admin Area =

1. In the admin area go to Plugins > Add New and search for "LinkGreen Site Integration"
2. Click install and then click activate

= To Setup The Plugin =

1. Find your LinkGreen API key (see instructions under FAQ).
2. In the WordPress admin area go to LinkGreen > Settings and then paste your API key

= How to Use the Shortcode =

1. Navigate to the post or page you would like to add the carousel to
2. Enter in the shortcode [linkgreen_carousel]
3. Customizations can be made for item image size, background, text colors and more.


== Frequently Asked Questions ==

= How do I find my LinkGreen API Key? =

1. Login to your LinkGreen account.
2. On the top right, select your username.
3. Select the "API Key" tab and select and copy it
